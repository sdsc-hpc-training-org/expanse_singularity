Prerequisites

The purpose of this tutorial is to enable you to use XSEDE's Expanse supercomputer at SDSC to run your jobs. You will need the following:

  • Basic familiarity with using Expanse
    • Expanse User Guide
    • Getting Started with Expanse tutorial
  • An account on Expanse; there are several options if you do not already have an account on Expanse. All of these will require that you first create a free account on the XSEDE User Portal
    • Currently, Expanse is unique in its status as an XSEDE system on which users may request a trial allocation to evaluate resources
    • If you are at a site with a Campus Champion, you can request a small amount of compute time
    • If your research is limited by the limited compute power you have in your research lab, you can request a Startup Allocation

Tutorials on how to get started with Singularity generally do not include details specific to running Singularity on XSEDE systems such as Expanse. In this tutorial, we will review how to access a compute node on Expanse and provide a simple example of how to run a Tensorflow job with Singularity on Expanse to help get you started.
