Prerequisites

The purpose of this tutorial is to enable you to use ACCESS's Expanse supercomputer at SDSC to run your jobs using Singularity. You will need the following:

  • Basic familiarity with using Expanse
    • Expanse User Guide
    • Getting Started with Expanse tutorial
  • An account on Expanse; there are several options if you do not already have an account on Expanse. All of these will require that you first create a free account on the ACCESS User Portal
    • Currently, Expanse is unique in its status as an XSEDE system on which users may request a trial allocation to evaluate resources
    • If your research is limited by the limited compute power you have in your research lab, you can request a ACCESS Explore Allocation

Tutorials on how to get started with Singularity generally do not include details specific to running Singularity on ACCESS systems such as Expanse.
